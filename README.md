# MyAntiCheat 🛡️

src/main/java/me/yourname/anticheat/

## Функції
- **Reach Check**: Блокує удари далі ніж 3.8 блоки.
- **Packet Based**: Працює на рівні пакетів для більшої точності.

## Як зібрати
1. Встановіть [Maven](https://apache.org).
2. Виконайте команду: `mvn clean package`.
3. Заберіть готовий `.jar` файл із папки `target/`.
